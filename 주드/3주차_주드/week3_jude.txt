## 1

# 홈화면
GET /users/{user-id}/home or
GET /users/home

=> 유저마다 홈화면을 다르게 보여주려면 /users/{user-id}/home
근데 보통 인증 토큰으로 주체 구분하기에 /users/home 쓴다함

# 내가 받은 미션
GET /users/{user-id}/home/missions?isAccepted=true

=> users/{user-id}가 먼저? home이 먼저? 고민함
IA에는 로그인 후 홈 인거같으니 유저정보를 먼저 썻음 /
이것도 인증 토큰 기반으로 사용자 식별하면 {user-id} 필요 x

# 마이페이지 리뷰작성? IA에선 미션에 리뷰작성 있어서 IA기준으로함

## 3

# 미션 목록 조회(진행중, 진행완료)
GET /users/{user-id}/missions?is_accepted=true&is_completed=false // 진행중
GET /users/{user-id}/missions?is_accepted=true&is_completed=true // 완료

=> 간단하게 받았냐, 완료했냐 정도의 쿼리로 구성했는데 
받았는데 (기간 때문에)종료된 미션등을 나타내려면 날짜 쿼리도 들어가야할듯
