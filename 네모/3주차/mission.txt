1. 홈 화면
[API Endpoint]
GET /users/{userId}/home
[Query String]
?regionName=안암동
[Path Variable]
userId
[Request Header]
Authorization: Bearer {accessToken}

2. 마이페이지 리뷰 작성
[API Endpoint]
POST /users/{userId}/review
[Path Variable]
userId
[Request Body]
{
	rating: 5,
	content: "맛있어요",
	image_url: [
		"aa",
		"bb"
	]
}
[Request Header]
Authorization: Bearer {accessToken}

3. 미션 목록 조회(진행 중, 진행 완료)
[API Endpoint]
GET /users/{usersId}/missions
[Query String]
?status=COMPLETED/IN_PROGRESS
[Path Variable]
userId
[Request Header]
Authorization: Bearer {accessToken}

4. 미션 성공 처리
[API Endpoint]
PATCH /users/{userId}/missions/{missionId}/complete
[Path Variable]
userId, missionId
[Request Header]
Authorization: Bearer {accessToken}
